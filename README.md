旅遊景點列表顯示

應用程式情境：
這是一個展示旅遊景點照片的 App。使用者可以在主畫面透過下拉選單（Spinner）改變照片的排列方式（直向清單、橫向清單、網格、瀑布流），點擊照片後會進入詳細介紹頁面。

功能重點：
- ⭐️切換多個Activity(畫面), 帶資料過去
- 旋轉螢幕時資料要存起來
- Spin View
- Recycle View（List View）
	- staggered
	- grid
	- 橫向
	- 直向
